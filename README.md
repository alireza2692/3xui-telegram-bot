# 3xui-telegram-bot

ربات فروش خودکار سرویس VPN برای تلگرام، متصل به پنل **3X-UI**. کاربران می‌توانند سرویس بخرند، تست رایگان بگیرند، کیف پول شارژ کنند، سرویس‌های خود را مدیریت و تمدید کنند — همه به‌صورت خودکار و بدون نیاز به دخالت دستی ادمین (به‌جز تایید پرداخت‌های کارت‌به‌کارت).

---

## ✨ امکانات

### برای کاربران
- 🛒 **خرید سرویس** با انتخاب حجم دلخواه، قیمت‌گذاری پلکانی (سه رنج قیمتی)
- 🏷️ پشتیبانی از **کد تخفیف**
- 🎁 **تست رایگان** (یک‌بار برای هر کاربر)
- 📋 **سرویس‌های من**: مشاهده حجم کل/مصرف‌شده/باقی‌مانده، تاریخ انقضا، لینک اشتراک
- 🔄 **تغییر لینک اتصال** بدون از دست رفتن حجم مصرف‌شده
- 🔁 **تمدید سرویس** (تمدید همیشه ۳۰ روز از لحظه تمدید تنظیم می‌شود)
- 💳 **کیف پول** با شارژ از طریق کارت‌به‌کارت (چند شماره کارت هم‌زمان قابل تنظیم)
- 🌍 پشتیبانی از **چند Inbound/سرور** — اگر بیش از یک Inbound تعریف شود، کاربر هنگام خرید سرور را انتخاب می‌کند
- 🔗 **سیستم دعوت (Referral)**: با اولین شارژ کیف پول کاربر دعوت‌شده، درصدی پاداش به دعوت‌کننده تعلق می‌گیرد
- 📚 **آموزش اتصال** بر اساس سیستم‌عامل (اندروید/iOS/ویندوز/مک) با لینک دانلود اپلیکیشن‌های پیشنهادی
- ⏰ **یادآوری خودکار** وقتی حجم یا زمان سرویس رو به اتمام است (فقط یک‌بار به ازای هر رویداد)
- 🆘 دکمه پشتیبانی مستقیم

### برای ادمین
- ✅ تایید/رد پرداخت با یک کلیک (از طریق دکمه‌های اینلاین)
- 👥 پشتیبانی از **چند ادمین** هم‌زمان
- 📊 `/stats` — آمار کامل: تعداد کاربران، سفارشات، درآمد کل/امروز، پرداخت‌های معلق، وضعیت سیستم‌ها
- 📢 `/broadcast` — ارسال پیام همگانی به تمام کاربران
- 🏷️ `/addcode` `/delcode` `/codes` — مدیریت کامل کدهای تخفیف
- 💰 `/setprice` — تغییر آنی قیمت‌ها و رنج‌های قیمتی بدون نیاز به ری‌استارت یا ویرایش کد
- 🔀 `/referral` `/trial` — فعال/غیرفعال کردن سیستم دعوت و تست رایگان
- ⏰ **یادآوری خودکار پرداخت‌های معلق** به ادمین (اگر بیش از ۲ ساعت بدون بررسی بماند)
- 📝 **کانال لاگ اختصاصی**: ثبت خودکار هر خرید، تمدید، شارژ کیف پول، تغییر لینک، رد پرداخت و پاداش دعوت

### زیرساخت و مدیریت
- 🐍 نوشته‌شده با **Python 3** و **aiogram 3.x** (async کامل)
- 🗄️ دیتابیس **SQLite** با ایندکس‌گذاری بهینه برای سرعت بالا
- 🌐 پشتیبانی از **پروکسی SOCKS5** برای دور زدن تحریم (اختیاری، برای سرورهای ایران)
- 🔌 اتصال به پنل صرفاً از طریق **API Token** (بدون نیاز به یوزرنیم/پسورد پنل)
- 🖥️ ابزار مدیریت خط‌فرمان اختصاصی: **`vpn-bot`**
- ⚙️ سرویس **systemd** با ری‌استارت خودکار در صورت کرش یا ری‌بوت سرور

## 🖥️ ابزار مدیریت `vpn-bot`

بعد از نصب، با اجرای دستور `vpn-bot` یک پنل کنترل کامل در ترمینال باز می‌شود:

```
╔══════════════════════════════════════════════════╗
║          🤖  VPN Sales Bot — Control Panel         ║
╚══════════════════════════════════════════════════╝
  Service status: ● RUNNING
──────────────────────────────────────────────────────
  1)  🟢 Start bot
  2)  🔴 Stop bot
  3)  🔄 Restart bot
  4)  📊 Service status
  5)  📜 Live logs
  6)  📄 Last 100 log lines
  7)  ⚙️  Edit settings (categorized menu)
  8)  💾 Create backup
  9)  ♻️  Restore from backup
  10) 📈 Quick database stats
  11) 📦 Update Python packages
  12) 🩺 Health check (panel, proxy, token)
  13) 🗑  Uninstall CLI & service
  0)  Exit
```

بخش **Edit settings** خودش دسته‌بندی‌شده است (شبیه منوی 3X-UI):

```
1) Bot & Admins
2) Mandatory Channel
3) Panel (3X-UI)
4) Inbounds / Servers
5) SOCKS5 Proxy
6) Log Channel
7) Payment Cards
8) Raw .env editor (پیشرفته)
```

هر بخش فقط مقادیر مربوط به خودش را نشان می‌دهد (توکن‌ها ماسک‌شده) و تغییر هر فیلد بدون نیاز به دیدن کل فایل `.env` انجام می‌شود.

دستورات سریع بدون منو هم در دسترس‌اند:
```bash
vpn-bot start | stop | restart | status | logs | backup | restore | health
```

## 📋 پیش‌نیازها

- یک سرور لینوکسی (Ubuntu/Debian توصیه می‌شود) با دسترسی root
- یک پنل **3X-UI** فعال با یک **API Token** ساخته‌شده
- توکن ربات تلگرام (از [@BotFather](https://t.me/BotFather))
- (اختیاری) پروکسی SOCKS5 محلی، اگر سرور شما مستقیم به تلگرام دسترسی ندارد

اسکریپت نصب خودش تمام پیش‌نیازهای نرم‌افزاری (Python، venv، sqlite3، nano و غیره) را بدون نیاز به دخالت شما نصب می‌کند.

## 🚀 نصب

فقط این یک دستور را در ترمینال سرور خود اجرا کنید:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/alireza2692/3xui-telegram-bot/main/install.sh)
```

> جای `USERNAME/REPO_NAME` را با آدرس مخزن گیت‌هاب خودتان جایگزین کنید.

اسکریپت به‌ترتیب:
1. می‌پرسد که نصب تازه می‌خواهید یا بازیابی از یک بکاپ قبلی
2. در حالت نصب تازه، سورس بات را از همین مخزن گیت‌هاب دانلود می‌کند
3. تمام تنظیمات لازم (توکن بات، پنل، Inbound ها، کارت‌ها، پروکسی و ...) را از شما می‌پرسد
4. پیش‌نیازهای سیستمی، virtual environment و پکیج‌های پایتون را نصب می‌کند
5. یک سرویس `systemd` به نام `vpnbot` می‌سازد (ری‌استارت خودکار در صورت کرش)
6. ابزار مدیریت `vpn-bot` را در `/usr/local/bin` نصب می‌کند
7. در پایان می‌پرسد که همین الان بات استارت شود یا نه

بعد از پایان نصب، برای مدیریت بات کافی است بزنید:
```bash
vpn-bot
```

## 🔄 آپدیت به نسخه جدید

اگر بات را قبلاً نصب کرده‌اید و می‌خواهید نسخه جدید کد را بگیرید، همان دستور نصب را دوباره اجرا کنید و در مرحله اول گزینه **Restore from backup نه**، بلکه در پرسش «Continue and overwrite files» پاسخ `y` را بدهید — فایل `.env` شما دست‌نخورده باقی می‌ماند و فقط کد آپدیت می‌شود (این رفتار به مسیر مشخص‌شده در اسکریپت بستگی دارد).

برای اطمینان کامل، همیشه قبل از آپدیت یک بکاپ بگیرید:
```bash
vpn-bot backup
```

## 💾 بکاپ و بازیابی

گرفتن بکاپ (شامل کد، دیتابیس و `.env`):
```bash
vpn-bot backup
```

بازیابی از یک بکاپ قبلی (از داخل منو یا مستقیم):
```bash
vpn-bot restore
```

بکاپ‌ها در پوشه `backups/` داخل مسیر نصب بات ذخیره می‌شوند.

## ⚙️ تنظیمات قابل تغییر بعد از نصب

از طریق `vpn-bot` → گزینه ۷ (Edit settings):
- توکن بات و لیست ادمین‌ها
- کانال اجباری عضویت
- آدرس/مسیر/توکن پنل
- افزودن، حذف یا جایگزینی کامل Inbound ها
- پروکسی SOCKS5
- کانال لاگ
- شماره کارت‌های پرداخت (افزودن/حذف/ویرایش)

از طریق دستورات تلگرام (فقط ادمین):
- `/setprice` — تغییر قیمت هر گیگابایت در هر رنج
- `/addcode` `/delcode` `/codes` — مدیریت کدهای تخفیف
- `/referral` `/trial` — روشن/خاموش کردن این دو سیستم
- `/broadcast` — پیام همگانی
- `/stats` — آمار کامل
